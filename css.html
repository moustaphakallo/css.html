Différences et similitudes entre let, var, et const

Similitudes

  • Déclaration de variables : Les trois ( let, var, const ) déclarent des variables.
  • Stockage de tout type de données : Ils stockent chaînes, nombres, objets, tableaux, etc.

Différences

let :
  • Portée de bloc (accessibles uniquement dans { ... }).
  • Ne peut pas être redéclaré dans une même portée.
  • Peut recevoir une nouvelle valeur.
var :
  • Possède une portée fonctionnelle.
  • Peut être redéclaré et réaffecté.
  • Est hoisté mais **non initialisé**.
const :
  • Portée de bloc (comme let).
  • Ne peut pas changer de valeur ni être redéclaré.
  • Doit être initialisé lors de sa déclaration.

Comprendre ces différences est crucial pour écrire du code JavaScript efficace !
