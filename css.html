Introduction à JavaScript

1️⃣ Histoire de JavaScript

JavaScript a été créé en 1995 par Brendan Eich chez Netscape pour rendre les pages web interactives.
Il a évolué avec AJAX (2005) pour le chargement dynamique, Node.js (2009) pour le côté serveur et des frameworks modernes comme React ou Vue.js.
L’arrivée de ES6 (2015) a apporté let, const, classes, modules, fonctions fléchées, rendant le code plus clair et moderne.
Aujourd’hui, JavaScript est incontournable pour le développement web front-end et back-end.
