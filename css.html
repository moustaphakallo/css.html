Exercice : Introduction à JavaScript

1️⃣ Histoire de JavaScript

Dans votre réponse, assurez-vous de préciser :

  • L’année et le contexte de la création de JavaScript
  • Le nom du créateur
  • L’évolution du langage (AJAX, Node.js, etc.)
  • L’importance de ES6 (2015) et ce qu’il a apporté
  • Pourquoi JavaScript est devenu un langage incontournable

2️⃣ Les types de données en JavaScript

Expliquez les types suivants, avec définition + exemple :

  • String
  • Number
  • Boolean
  • Undefined
  • Null
  • Symbol
  • BigInt
  • Object
  • Array
  • Function

3️⃣ Différences entre var, let et const

Dans votre explication, mentionnez :

  • Si la variable peut être modifiée
  • Si elle peut être redéclarée
  • Le type de portée (scope)
  • Laquelle est recommandée aujourd’hui et pourquoi

// Exemple var var a = 10; a = 20; // Exemple let let b = 10; b = 20; // Exemple const const c = 10; // c = 20; // erreur
