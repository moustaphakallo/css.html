History of JavaScript

1️⃣ History of JavaScript

JavaScript was created in 1995 by Brendan Eich at Netscape to make web pages interactive.

It evolved with AJAX (2005), Node.js (2009), and modern frameworks like React and Vue.js.

The arrival of ES6 (2015) revolutionized the language: let, const, classes, modules, arrow functions.

Today, JavaScript is essential for both front-end and back-end development.

ES1 — 1997 ECMAScript 1

First standardized version of JavaScript.

ES2 — 1998 ECMAScript 2

Minor update aligned with ISO/IEC.

ES3 — 1999 ECMAScript 3

Added regex, error handling, and new operators.

ES4 ECMAScript 4

Never released (disagreements). Basis for ES6 ideas.

ES5 — 2009 ECMAScript 5

Modern methods (map, filter…), JSON support added.

ES6 — 2015 ECMAScript 6

Classes, modules, promises, let/const, arrow functions.

ES7 — 2016 ECMAScript 7

includes() added, exponentiation operator **.

ES8 — 2017 ECMAScript 8

Async/await, Object.entries, improved string handling.

ES9 — 2018 ECMAScript 9

Async/await and regex improvements.

ES10 — 2019 ECMAScript 10

flat(), flatMap(), Object.fromEntries.

ES11 — 2020 ECMAScript 11

BigInt, allSettled, ?? and optional chaining ?.

ES12 — 2021 ECMAScript 12

replaceAll, improved modules, private fields.

ES13 — 2022 ECMAScript 13

Optimizations, new APIs and improvements.
