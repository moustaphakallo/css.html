Histoire de JavaScript

1️⃣ Histoire de JavaScript

JavaScript a été créé en 1995 par Brendan Eich chez Netscape pour rendre les pages web interactives.

Il a évolué avec AJAX (2005), Node.js (2009) et des frameworks modernes comme React ou Vue.js.

L’arrivée de ES6 (2015) a révolutionné le langage : let, const, classes, modules, fonctions fléchées.

Aujourd’hui, JavaScript est indispensable en front-end et back-end.

ES1 — 1997 ECMAScript 1

Première version normalisée de JavaScript.

ES2 — 1998 ECMAScript 2

Mise à jour mineure alignée sur ISO/IEC.

ES3 — 1999 ECMAScript 3

Ajout regex, gestion d’erreurs, nouveaux opérateurs.

ES4 ECMAScript 4

Jamais publié (désaccords). Base d'idées pour ES6.

ES5 — 2009 ECMAScript 5

Méthodes modernes (map, filter…), JSON intégré.

ES6 — 2015 ECMAScript 6

Classes, modules, promesses, let/const, flèches.

ES7 — 2016 ECMAScript 7

Ajout de includes(), exponentiation **.

ES8 — 2017 ECMAScript 8

Async/await, Object.entries, amélioration chaînes.

ES9 — 2018 ECMAScript 9

Améliorations async/await et regex.

ES10 — 2019 ECMAScript 10

flat(), flatMap(), Object.fromEntries.

ES11 — 2020 ECMAScript 11

BigInt, allSettled, ?? et opérateur ?.

ES12 — 2021 ECMAScript 12

replaceAll, modules améliorés, champs privés.

ES13 — 2022 ECMAScript 13

Optimisations, nouvelles APIs et améliorations.
